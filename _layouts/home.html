Where Ingenuity Meets Community

At HoodyHerd, students learn to code and improve their problem solving skills by developing 2D games and mobile apps. As they develop coding expertise, they work to build software that benefits their communities.



Our Tracks:

GODOT


Learn to code and develop 2D games using the Godot Game Engine. By the end of this track, students will have general fluency with the game engine, and will have created multiple games they could add to their portfolios.

THUNKABLE


Using Thunkable, students learn how to develop cross-platform mobile apps that can be used on Android and iOS devices.

PYTHON


For members of the Herd who want the more authentic coding experience, and want to learn a language widely used in industry, Python will be used to create standalone applicatons.
